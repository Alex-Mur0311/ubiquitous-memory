# Музыкальный сайт
![alt text](assets/images/preview.png)

**MusicHouse** – это современный одностраничный сайт (лендинг) для любителей музыки. Проект демонстрирует навыки адаптивной вёрстки, работу с CSS‑переменными (светлая/тёмная тема), динамическую генерацию контента на JavaScript, валидацию формы и сохранение пользовательских настроек в `localStorage`.

## Цель проекта

Разработать полностью интерактивный музыкальный лендинг, соответствующий следующим критериям:

- Семантическая HTML‑структура (`<header>`, `<main>`, `<section>`, `<footer>`).
- Обязательные блоки: шапка с навигацией, карточки треков (каталог), форма обратной связи.
- Использование **Flexbox** и **Grid** для построения сетки карточек и преимуществ.
- Реализация кнопки **«Показать еще»** – динамическое добавление новых треков.
- Реализация **переключения светлой/тёмной темы** с сохранением выбора в `localStorage`.
- Валидация полей формы (проверка email, обязательность полей) и вывод данных в `console.log` при отправке.
- Адаптивность под мобильные устройства, планшеты и десктопы.
- Публикация на **GitHub Pages** и наличие подробного `README.md`.
